Image Preview:

How to use this application:

1. Select an image using the file input above.

2. The image will be displayed in the preview area.

3. To analyze the image for pig diseases, you would need to:

  • Install the required packages: Flask, TensorFlow, Pillow, NumPy
  • Copy the detect.tflite model to the models directory
  • Run the Flask application with: python app.py

For now, this simple viewer allows you to preview images while you work on setting up the dependencies.
